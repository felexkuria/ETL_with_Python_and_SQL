#### Description:
In this course, Jennifer Ebe, a data engineer with over 5 years of experience, will teach you how to build systems that gather data from various sources, transform it, and store it so you can derive insights effectively. You will learn how to use Python to extract and transform data, explore data using pandas and SQL, and load data into your target system. The course includes hands-on challenges to help you practice what you learn.
